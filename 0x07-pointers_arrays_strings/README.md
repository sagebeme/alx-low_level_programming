0x07. Pointers, arrays, strings (cont.)
========================================

This project continues **pointers, arrays, and strings** with more advanced tasks: memset, memcpy, strchr, strspn, strpbrk, strstr, print chessboard, or similar. You practice pointer arithmetic and memory-oriented functions.

-----

**Task numbers:** 0, 1, 2, 3, 4, 5, 7, 8, 100
