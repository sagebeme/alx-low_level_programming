0x1E. Search algorithms
=======================

This project practices **search algorithms** in C: linear search and binary search (and possibly jump search or interpolation search). You implement search in arrays and analyze time/space complexity (Big O notation).

Tasks
-----

**Task numbers:** 0, 1, 2, 3, 4, 5, 6, 100, 101, 102, 103, 104, 105, 106, 107, 108

### 0. Linear search

mandatory

Write a function that searches for a value in an array of integers using the Linear search algorithm. Return the first index where value is located, or -1 if not present or if array is NULL.

**Repo:**

-   GitHub repository: `alx-low_level_programming`
-   Directory: `0x1E-search_algorithms`
-   File: `0-linear.c`
