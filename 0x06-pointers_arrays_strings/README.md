0x06. Pointers, arrays, strings (cont.)
========================================

This project continues **pointers, arrays, and strings**: more string operations (e.g. copy, concatenate, compare), pointer arithmetic, and array-of-strings. You build on 0x05 with more complex functions.

Tasks
-----

**Task numbers:** 0, 1, 2, 3, 4, 5, 6, 7, 100, 101, 102, 103, 104

### 0. strcat

mandatory

Write a function that concatenates two strings. The destination string will be modified.
