0x08. Recursion
===============

This project practices **recursion** in C: functions that call themselves (e.g. factorial, power, sqrt, palindrome, wildcmp). You implement recursive solutions and understand base cases and recursive cases.

Tasks
-----

**Task numbers:** 0, 1, 2, 3, 4, 5, 6, 100, 101

mandatory

Write a function that prints a string, followed by a new line, using recursion (like puts but recursive).
